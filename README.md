# Egg Mix

> Egg Mix 是一套基于 Laravel Mix 和 Egg 的后端渲染多页应用工程化脚手架。

## Development

```
npm install 

npm run watch

npm run dev // running in http://localhost:7001

npm run build // build static files
```
